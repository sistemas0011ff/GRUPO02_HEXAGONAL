🚀 Resumen de Conceptos de TypeScript

💎 Tipos de Datos Primitivos

  • 📝 String: Textos con comillas dobles, simples o plantillas literales (``)
  • 🔢 Number: Único tipo para todos los números (enteros, decimales, notación científica)
  • 🔣 Sistemas numéricos: Binario (0b), Octal (0o), Hexadecimal (0x)
  • 🔍 Symbol: Valores únicos e inmutables

📦 Tipos Compuestos

  • 📋 Array: Colecciones de datos del mismo tipo o tipos mixtos
  • 🏷️ Objetos: Definidos con propiedades específicas
  • 📏 Tuplas: Arrays con número fijo de elementos de tipos específicos
  • ❓ any/unknown: Tipos flexibles para valores dinámicos

🔧 Tipos Avanzados

  • 🏗️ Type Alias: Definición de tipos personalizados reutilizables
  • 🔀 Union Types: Combinación de tipos con operador |
  • 🔄 Intersection Types: Combinación de tipos con operador &

⚙️ Funciones

  • 📞 Callbacks: Funciones pasadas como argumentos a otras funciones
  • ➡️ Arrow Functions: Sintaxis simplificada para funciones

🧩 Programación Orientada a Objetos

  • 📄 Interfaces: Contratos que definen estructuras
  • 🏛️ Clases: Implementación de interfaces con propiedades y métodos
  • 🔒 Encapsulación: Control de acceso con private, get/set
  • 👨‍👦 Herencia: Extensión de clases base a clases derivadas
  • 👨‍👦‍👦 Herencia múltiple: Clases que extienden a otras que ya son extensiones

🧪 Genéricos

  • 🔄 Clases genéricas: Componentes reutilizables con tipos flexibles
  • 🔄👨‍👦 Herencia de genéricos: Extensión de clases genéricas
  • 🔄👨‍👦‍👦 Múltiple herencia genérica: Clases genéricas que extienden a otras genéricas
  • 🔣 Genéricos con múltiples parámetros: <T, U> para usar varios tipos
